# Mobile br-email

Aplicativo Expo/React Native pronto para Android e iOS.

## Build
- Android APK: `npm run build:apk`
- iOS App Store ready: `npm run build:ios`

## Segurança
- Chaves privadas ficam no `expo-secure-store` (armazenamento seguro do dispositivo).
- O app só envia chaves públicas para o backend.
