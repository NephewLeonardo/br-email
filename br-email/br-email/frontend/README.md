# Frontend web br-email

Interface web minimalista em HTML/CSS/JavaScript usando Vite para build.

## Fluxo
1. Registro gera chaves localmente no navegador.
2. Apenas chaves públicas são enviadas para a API.
3. A caixa de entrada mostra envelopes criptografados armazenados no backend.
4. A descriptografia ocorre apenas no cliente.
